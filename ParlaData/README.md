# ParlaData

**Visualização intuitiva das votações da Câmara dos Deputados**

Plataforma web que transforma os dados abertos da Câmara dos Deputados em informações claras, acessíveis e visualmente atrativas, facilitando a fiscalização cidadã, o engajamento cívico e o acompanhamento de pautas legislativas.

---

## ✨ Funcionalidades

- Cadastro e login de usuários
- Recuperação de senha por e-mail
- Consulta de pautas e votações da Câmara dos Deputados
- Filtros avançados (tema, partido, estado, deputado, período)
- Gráficos interativos (distribuição de votos, presença/ausência, etc.)
- Interface responsiva e amigável (mobile-first)
- Integração com API oficial de Dados Abertos da Câmara

---

## 🛠 Tecnologias Utilizadas

**Frontend:**
- React + Vite
- CSS Modules / Tailwind (opcional)
- Chart.js ou Recharts (gráficos)

**Backend:**
- Node.js + Express
- MySQL
- bcryptjs (segurança de senhas)
- Nodemailer (recuperação de senha)

**Banco de Dados:**
- MySQL 8

**Containerização:**
- Docker + Docker Compose
